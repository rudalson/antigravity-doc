# 시작하기

## 다운로드

Antigravity IDE를 다운로드하려면 [antigravity.google/download](https://antigravity.google/download)를 방문하십시오.

<b>지원 플랫폼 및 최소 버전:</b>
- **macOS**: Apple 보안 업데이트 지원을 받는 macOS 버전. 일반적으로 현재 버전과 이전 두 개 버전입니다. 최소 버전 12 (Monterey), X86은 지원되지 않음
- **Windows**: Windows 10 (64비트)
- **Linux**: glibc >= 2.28, glibcxx >= 3.4.25 (예: Ubuntu 20, Debian 10, Fedora 36, RHEL 8)

애플리케이션은 업데이트가 가능할 때 메시지를 표시합니다.
