# 프롬프트 입력 및 상호작용

기본 상호작용 패턴, 여러 줄 입력 워크플로, 세션 중단 제어, 터미널 미디어 붙여넣기를 마스터하세요.

## 프롬프트 창 

Antigravity CLI는 터미널 화면 하단에 고정된 프롬프트 패널을 제공합니다. 이 패널은 일반 사용자 입력, 여러 줄 스크립트, 미디어 직접 붙여넣기를 처리합니다.

```text
───────────────────────────────────────────────────────────────────────────
> 여기에 다음 엔지니어링 작업을 입력하세요...
───────────────────────────────────────────────────────────────────────────
```

### 프롬프트 제출

에이전트 턴을 시작하려면 프롬프트 패널에 지침을 입력하고 `Enter`를 누릅니다. 에이전트는 즉시 현재 디렉터리 작업 공간을 분석하고, 필요한 설정을 읽고, 실행 계획을 수립하기 시작합니다.

### 실행 중인 세션 중단

에이전트가 원치 않는 작업을 시작하거나 명령어 실행 중에 무한 루프에 빠진 경우, `Esc`를 눌러 즉시 세션을 중단할 수 있습니다.

## 여러 줄 입력 

복잡한 지시사항, 구조화된 테스트 시나리오 또는 여러 문단으로 구성된 명령을 작성할 때는 내장된 여러 줄 입력 기능을 사용합니다.

### 줄바꿈 빠른 입력

- **기본**: 프롬프트 창을 제출하지 않고 줄바꿈만 추가하려면 `Shift+Enter` 또는 `ctrl+j`를 누릅니다.
- **macOS 터미널 대체 단축키**: 기본적으로 `Shift+Enter`를 전달하지 않는 Apple Terminal을 사용하는 경우, `Option+Enter`를 누릅니다. 이때 터미널 설정 프로필에서 **Option을 메타(Meta) 키로 사용**이 체크되어 있는지 확인하세요.
- **역슬래시를 통한 줄바꿈**: 줄 끝에 역슬래시 `\`를 입력하고 `Enter`를 누릅니다. CLI가 자동으로 역슬래시를 제거하고 줄바꿈을 추가합니다.

### `$EDITOR`에서 프롬프트 편집

기본 개발 에디터에서 긴 프롬프트 초안을 작성하거나 편집하려면 다음과 같이 하십시오.

1. 빈 프롬프트 패널에서 `ctrl+g`를 누릅니다.
2. CLI가 시스템의 기본 텍스트 에디터(예: `vim`, `nano`, `code` 등. `/config` 또는 환경 변수 `$EDITOR`를 통해 설정됨)를 실행합니다.
3. 임시 에디터 버퍼에 여러 줄 지침의 초안을 작성합니다.
4. 에디터를 저장하고 종료합니다. CLI가 편집된 버퍼의 내용을 터미널 프롬프트에 자동으로 다시 가져옵니다.

## 미디어 첨부

Antigravity CLI는 시스템 클립보드에서 다양한 미디어 형식을 직접 붙여넣는 것을 지원합니다. 프롬프트 패널 내에서 `ctrl+v`(또는 터미널 자체 붙여넣기 기능)를 눌러 스크린샷 목업이나 비디오 녹화본을 첨부할 수 있습니다.

### 지원하는 파일 형식

- **이미지**: PNG, JPEG, GIF, WebP, BMP, TIFF, SVG.
- **동영상**: MP4, MOV, WebM, AVI.
