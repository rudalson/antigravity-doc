# 설정

## Antigravity 2.0 설정 

Antigravity 2.0은 개발 환경을 세밀하게 제어할 수 있도록 계층형 설정 아키텍처를 제공합니다. 강력한 보안과 유연한 작업 공간 구성을 보장하기 위해 설정은 전역 애플리케이션 기본 설정과 격리된 프로젝트 수준의 경계로 나뉩니다.

### 설정에 접근하는 방법

다음 방법을 사용하여 Antigravity 2.0의 설정 패널을 열 수 있습니다:

* **키보드 단축키**: 애플리케이션 내 활성화된 화면에서 `Cmd + ,`를 누릅니다.  
* **사이드바 탐색**: 왼쪽 사이드바 하단의 Settings(설정)를 클릭합니다.  
* **프로젝트 설정**: 특정 프로젝트 옆에 위치한 기어 아이콘을 클릭합니다.

### 네 가지 설정 카테고리

구성을 깔끔하고 격리된 상태로 유지하기 위해 설정은 네 가지의 고유한 범위로 구성됩니다:

**1. 전역 설정 (Global Settings)**

모든 대상에 적용되는 전역 설정입니다:

* 계정 설정 (Account Settings): 인증 세션을 관리하고 텔레메트리(모델 개선을 위한 상호작용 로그 공유 활성화/비활성화)를 전환합니다.  
* 전역 권한 (Global Permissions): 모든 대화에 적용되는 중앙 집중식 기본 도구 경계 설정입니다.  
* 외관 (Appearance): 시각적 테마 및 패널 레이아웃을 맞춤 구성합니다.  
* 브라우저 통합 (Browser Integration): 에이전트가 웹 화면과 상호작용하는 방식을 구성합니다.  
* 모델 사용 (Model Usage): 기본 추론 모델을 선택하고 구성합니다.  
* 사용자 정의 (Customizations): MCP(Model Context Protocol) 서버, 맞춤형 스킬, "Build with Google" 플러그인을 관리합니다.

**2. 프로젝트 설정 (Project Settings)**

특정 프로젝트 범위 내에만 독점적으로 적용되는 설정입니다:

* 폴더 (Folders): 프로젝트와 연결된 로컬 폴더 목록을 정의합니다. Antigravity는 대화 대상을 처리하기 위해 이 폴더들의 Git 구성을 자동으로 감지합니다:  
  * Local(로컬): 새 대화 보기에서 이를 선택하여 기존 폴더에서 직접 작업합니다.  
  * Worktree(워크트리): 폴더 내에서 새 워크트리를 시작하려면 이를 선택합니다. (참고: 폴더에 Git이 설정되어 있지 않은 경우, 기존의 로컬 폴더가 대신 사용됩니다).  
* 에이전트 설정 (Agent Settings): 프로젝트 전용 에이전트 동작을 구성합니다:  
  * 터미널 실행 정책 (Terminal Execution Policy): 에이전트가 셸 명령을 실행하는 방식을 제어합니다.  
  * 폴더 외 파일 액세스 정책 (Outside of Folder File Access Policy): 프로젝트 경계 외부의 파일에 에이전트가 액세스하는 방식을 정의합니다 (항상 허용, 항상 확인, 또는 항상 거부).  
  * 샌드박스 모드 (Sandbox Mode): 사용자 정의 보안 사전 설정 내에서 터미널 샌드박스 컨테이너의 활성화 여부를 전환합니다.  
* 프로젝트 수준 권한 (Project-level Permissions): 프로젝트 수준에서 권한을 구성합니다. 에이전트와 상호작용함에 따라 권한 요청이 누적되며, 이를 프로젝트 권한에 자동으로 추가할 수 있습니다.  
* 사용자 정의 (Customizations): 전역 사용자 정의 및 프로젝트 전용 사용자 정의 모두에서 파생됩니다. 프로젝트에 추가된 각 폴더에서 생성된 모든 스킬을 볼 수 있습니다.

**3. 프로젝트 외 대화 (Conversations Outside of a Project)**

프로젝트 외부에서도 대화(단독 대화)를 시작할 수 있습니다:

* 동작: 이 대화들은 구성 가능한 폴더가 없으며, 대신 로컬 임시(scratch) 디렉터리에서 실행됩니다.  
* 설정: 프로젝트와 유사하게 자체 설정(터미널 실행, 파일 액세스 정책, 권한 등)을 가질 수 있지만, 프로젝트 구조와 독립적으로 작동합니다.

**4. 기타 설정 (Miscellaneous)**

* 단축키 (Shortcuts): 키보드 단축키 구성을 확인하고 맞춤 설정합니다.  
* 피드백 (Feedback): 팀에 직접 보고서를 전송할 수 있는 피드백 양식에 액세스합니다.

## 데이터 수집 설정

"Enable Telemetry"(텔레메트리 활성화) 설정은 설정 패널의 "Account"(계정) 섹션에서 찾을 수 있습니다. 이 설정을 켜면 Antigravity 및 Antigravity를 지원하는 모델을 평가, 개발 및 개선하기 위해 상호작용 데이터를 수집합니다.
