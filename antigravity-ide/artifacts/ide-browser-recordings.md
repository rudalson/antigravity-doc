# 브라우저 녹화 (Browser Recordings)

브라우저 서브에이전트가 브라우저를 조작할 때마다 사용자의 검토를 위해 에이전트의 작업에 대한 녹화본 생성을 선택할 수 있습니다. 녹화가 가능한 경우 브라우저 단계 UI 하단에서 이 비디오 재생을 시청할 수 있습니다.

![브라우저 녹화 캡처 도구](assets/image/docs/artifacts/browser-recording-capture.png)

모든 브라우저 녹화는 사용자의 검토를 위해 녹화 아티팩트로도 저장됩니다. 이 보기에서는 브라우저 에이전트의 작업을 루프로 반복하여 보여줍니다.

![브라우저 녹화 아티팩트](assets/image/docs/artifacts/browser-recording-artifact.png)
