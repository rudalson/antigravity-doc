# AGY CLI 사용하기

### 설정 

Antigravity CLI는 작업 공간의 동작, 안전 제한, 에디터 선호도, 시각적 스타일 및 성능을 맞춤 설정할 수 있는 유연한 설정 시스템을 제공합니다.

* **설정 파일**: 일반 JSON 파일인 `~/.gemini/antigravity-cli/settings.json`에 저장됩니다.  
* **설정 패널**: `/config` 또는 `/settings`를 입력하면 사용 가능한 모든 옵션이 나열된 전체 화면 오버레이 메뉴가 열립니다.  
  * 설정을 선택하면 해당 설정의 옵션 목록이나 텍스트 입력 필드가 열립니다.  
  * 선택한 내용은 디스크에 즉시 저장되며 메인 목록으로 돌아갑니다.  
* **재정의**: 특정 설정은 실행 시 CLI 플래그(예: `--sandbox` 또는 `--dangerously-skip-permissions`)를 통해 재정의할 수 있습니다.  
  * 설정 메뉴에는 재정의가 어디서 시작되었는지 나타내는 표시가 나타납니다(예: *Sandbox Mode가 `--sandbox`에 의해 재정의되어 켜짐*).  
  * 디스크에 있는 영구 설정을 편집할 수는 있지만, 현재 세션은 재시작할 때까지 명령줄 재정의를 강제로 적용합니다.

### 빠른 팁 

| 동작/기능 | 팁/명령어 |
| :--- | :--- |
| **파일 경로 자동 완성** | `@`을 입력하면 경로 추천이 활성화됩니다. |
| **프롬프트 비우기** | 프롬프트 입력창을 비우려면 `esc esc`를 입력합니다 (스트리밍이 진행 중이지 않을 때). |
| **터미널 명령어** | 프롬프트 시작 부분에 `!`를 사용하여 터미널 명령어를 직접 실행할 수 있습니다. |
| **도움말** | 도움말을 보고 모든 슬래시 명령어 목록을 확인하려면 `?`를 입력합니다. |
| **도구 호출 노이즈 줄이기** | 수많은 도구 호출로 인한 출력을 최소화하려면 `/config`에서 상세도(verbosity)를 **low**로 설정합니다. |
| **권한 관리** | `/config` 또는 `/permissions`를 통해 권한을 제어합니다. |
| **대화 되감기** | 대화 기록을 되감으려면 `/rewind` 또는 `/undo`를 사용합니다. |
| **대화 분기 (Fork)** | 별도의 작업 공간을 만들고 이전 시점부터 대화를 분기하려면 `/fork`를 사용합니다. |
| **대화 초기화** | 프롬프트를 비우고 새 대화 세션을 시작하려면 `/clear`를 사용합니다. |
| **대화 재개** | 이전 대화 로그를 나열하고 재개하려면 `/resume`를 사용합니다. |
| **자동 저장 후 재개** | CLI를 닫을 때, 해당 특정 세션을 재개하는 데 필요한 정확한 명령어가 자동으로 출력됩니다. |

### 단축키 

AGY CLI는 사용자 정의 단축키를 지원합니다. `/keybindings`를 입력하거나 JSON 파일을 직접 수정하여 단축키를 편집할 수 있습니다.

* **파일 위치**: `~/.gemini/antigravity-cli/keybindings.json`.  
* **초기화**: 기본값으로 재설정하려면 `keybindings.json` 파일을 삭제합니다.  

**기본 단축키**

| 동작/명령어 | 키 | 목적 |
| :--- | :--- | :--- |
| **TUI 화면 지우기** | `ctrl+l` | 터미널 출력 지우기 |
| **입력 / 제출** | `enter` | 프롬프트 또는 선택 항목 제출 |
| **이스케이프 / 취소** | `ctrl+c`, `esc` | 스트림 중지, 메뉴 닫기 또는 프롬프트 비우기 |
| **CLI 종료** | `ctrl+d` | CLI TUI 세션 종료 |
| **CLI 일시 중단** | `ctrl+z` | CLI 세션을 터미널 백그라운드로 보냄 |
| **명령어 편집** | `e` | 제안된 터미널 명령어를 편집할 에디터 열기 |
| **아니오 확인** | `n` | 터미널 명령어 실행 거절 |
| **예 확인** | `y` | 터미널 명령어 실행 승인 |
| **에디터 열기** | `ctrl+g` | 기본 셸 에디터 내에서 프롬프트 편집 |
| **텍스트 붙여넣기** | `ctrl+v` | 클립보드에서 텍스트 붙여넣기 |
| **텍스트 편집 다시 실행** | `ctrl+shift+z` | 마지막으로 실행 취소한 텍스트 변경 사항 다시 실행 |
| **텍스트 편집 실행 취소** | `ctrl+_`, `ctrl+shift+-` | 마지막 텍스트 변경 사항 실행 취소 |
| **복사 (Yank)** | `ctrl+y` | 선택한 텍스트 복사/Yank |
| **아래로 이동** | `down` | 메뉴 목록에서 아래로 스크롤 |
| **맨 아래로 이동** | `ctrl+end` | TUI 뷰를 맨 아래로 바로 이동 |
| **맨 위로 이동** | `ctrl+home` | TUI 뷰를 맨 위로 바로 이동 |
| **왼쪽으로 이동** | `left` | 프롬프트 커서를 왼쪽으로 이동 |
| **페이지 다운** | `pgdown`, `shift+down` | TUI 페이지를 아래로 스크롤 |
| **페이지 업** | `pgup`, `shift+up` | TUI 페이지를 위로 스크롤 |
| **오른쪽으로 이동** | `right` | 프롬프트 커서를 오른쪽으로 이동 |
| **탭 / 포커스** | `tab` | 선택 항목 자동 완성 또는 컴포넌트 포커스 전환 |
| **위로 이동** | `up` | 메뉴 목록에서 위로 스크롤 |
| **줄바꿈 입력** | `alt+enter`, `ctrl+j`, `shift+enter` | 제출하지 않고 프롬프트에 줄바꿈 추가 |

JSON 파일에서 단일 동작을 여러 단축키에 매핑할 수 있습니다. 단축키를 비활성화하려면 목록을 비워둡니다 (예: `[]`). 파일 형식이 잘못된 경우 CLI는 유효한 부분을 사용하고 손상된 동작에 대해서는 기본값으로 대체합니다.
