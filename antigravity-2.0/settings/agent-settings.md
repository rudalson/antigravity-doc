# 에이전트 설정

### 터미널 명령 자동 실행

에이전트가 생성된 셸 명령을 실행하는 방식을 제어합니다:
* **Request Review**(검토 요청): 에이전트가 터미널 명령을 자동으로 실행하지 않습니다 (구성 가능한 허용 목록에 명시적으로 추가된 명령은 제외).
* **Always Proceed**(항상 진행): 에이전트가 사용자에게 묻지 않고 명령을 자동으로 실행합니다 (구성 가능한 거부 목록에 명시적으로 추가된 명령은 제외).

### 에이전트의 비워크스페이스 파일 액세스

에이전트가 활성 프로젝트 폴더 외부의 파일을 보고 편집할 수 있도록 허용합니다. 
* 기본적으로 에이전트는 프로젝트 내부의 폴더와 애플리케이션의 로컬 앱 데이터 디렉터리인 `~/.gemini/antigravity/`(아티팩트, 지식 항목 등이 포함됨)에만 액세스할 수 있습니다.
* 이 경계를 강제 적용함으로써 로컬에 있는 민감한 데이터를 보호합니다. 비워크스페이스 액세스 활성화 시 주의하시기 바랍니다.
