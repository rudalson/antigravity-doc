# Antigravity 2.0

## 개요

에이전트 우선(Agent-first) 시대로 더 깊이 진입함에 따라, Antigravity는 개발자에게 더 큰 힘을 실어주기 위해 진화하고 있습니다. 복잡한 지식 및 코딩 작업을 수행하는 AI 에이전트를 관리하기 위해 특별히 제작된 단독 데스크톱 애플리케이션인 Antigravity 2.0을 소개합니다.

### Antigravity 2.0이란 무엇인가요?

Antigravity 2.0은 AI 에이전트의 중앙 커맨드 센터 역할을 하며, 에이전트의 활동을 시작, 모니터링 및 오케스트레이션할 수 있는 통합 플랫폼을 제공합니다. 이전 버전인 Agent Manager와 달리, Antigravity 2.0은 IDE와 독립적으로 작동하는 단독 애플리케이션입니다.

이 인터페이스 내에서 에이전트를 동기 및 비동기 방식으로 다음과 같이 오케스트레이션할 수 있습니다:

* 시스템 명령 실행  
* 파일 읽기/쓰기 작업 수행  
* 웹 검색 수행  
* 스킬 및 MCP 서버를 통한 외부 도구와의 통합  
* 서브에이전트 관리  
* Chrome과의 상호작용  
* 아티팩트 / 구현 계획 생성

심층 연구를 수행하든 새로운 애플리케이션을 빌드하든, Antigravity는 모든 유형의 지식 작업을 위한 효율적이고 능률적인 인터페이스를 제공합니다.
