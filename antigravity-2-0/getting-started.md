# Antigravity 2.0 시작하기

### 다운로드 

Google Antigravity 2.0을 다운로드하려면 [antigravity.google/download](https://antigravity.google/download)를 방문하세요. 

* **macOS**: Apple 보안 업데이트 지원을 받는 macOS 버전입니다. 일반적으로 현재 버전 및 이전 두 버전이 해당합니다. 최소 버전은 12 (Monterey)이며, X86은 지원되지 않습니다.
* **Windows**: Windows 10 (64비트)
* **Linux**: glibc >= 2.28, glibcxx >= 3.4.25 (예: Ubuntu 20, Debian 10, Fedora 36, RHEL 8)

### 설치

Antigravity를 “둘 다 유지(Keep Both)”할지 “대체(Replace)”할지 묻는 알림이 표시될 수 있으며, 이 경우 “대체”를 선택하세요. 설치 과정에서 원하는 경우 IDE를 재설치하라는 메시지가 표시됩니다. 지금 설치하지 않고 나중에 다시 다운로드하려면 [여기](/download)에서 다운로드할 수 있습니다.

### 프로젝트 생성하기

에이전트는 프로젝트 내에서 작업하며, 프로젝트는 에이전트가 액세스할 수 있는 폴더 및 리포지토리의 경계를 정의합니다.

1. **왼쪽 사이드바**에서 **'+' 아이콘이 있는 폴더**를 클릭합니다.
2. **"New Project"**(새 프로젝트)를 클릭합니다.
3. 하나 이상의 로컬 폴더 또는 Git 리포지토리를 연결하려면 **Add Folder**(폴더 추가)를 클릭합니다. 여러 폴더를 추가하면 에이전트에게 완전한 크로스 리포지토리(cross-repository) 컨텍스트가 제공됩니다.
4. **Create**(생성)를 클릭합니다.
5. *(선택 사항)* 프로젝트 설정을 구성합니다. 각 프로젝트는 에이전트가 준수하는 고유한 격리된 설정 및 보안 정책을 유지합니다.


### 에이전트 시작하기

프로젝트가 생성되면 에이전트를 가동(spawn)하여 작업을 시작할 수 있습니다.

1. 채팅 입력창에 목표 또는 명령(예: "새 기능 추가를 도와줘")을 입력하고 **Enter**를 누릅니다.
2. 설정 모달에서 에이전트를 실행할 **모드**를 선택합니다:
   * **Local Mode**(로컬 모드): 에이전트가 활성화된 폴더에서 직접 작동합니다.
   * **New Worktree Mode**(새 워크트리 모드): 에이전트가 격리된 Git 워크트리에서 작동합니다.

### 기본 탐색

| 작업 | macOS | Windows / Linux |
| :--- | :--- | :--- |
| **대화 선택기 열기** | `⌘K` | `Ctrl + K` |
| **파일 검색 열기** | `⌘P` | `Ctrl + P` |
| **입력창 포커스** | `⌘L` | `Ctrl + L` |
| **새 대화** | `⌘N` | `Ctrl + N` |
| **다음/이전 대화** | `⌥ Up / Down` | `Alt + Up / Down` |

### 슬래시 명령어

* `/goal`: 중간에 사용자에게 입력을 요청하지 않고, 지정된 작업이 완전히 끝날 때까지 실행합니다.  
* `/grill-me`: 구현을 시작하기 전에 계획의 구체적인 세부 사항을 조율하기 위해 사용자에게 질문을 던집니다.  
* `/schedule`: 미래의 특정 시점에 일회성 타이머로 명령을 실행하거나 반복 일정(예약된 작업 이용)에 따라 명령을 실행합니다.  
* `/browser`: 에이전트가 브라우저를 언제 사용해야 하는지 정확히 판단하는 능력이 여전히 부족하다는 피드백을 수렴했습니다. 이에 따라 현재로서는 명시적인 슬래시 명령어를 통해 이러한 동작을 제어하도록 설정했습니다. 이 명령어를 사용하면 에이전트가 브라우저 기본 프리미티브(primitives)를 성실히 사용합니다. 이를 위해서는 Google Chrome이 설치되어 있어야 하며, 디버깅 세션을 시작할 수 있도록 사용자가 Google Chrome에서 권한을 제공해야 합니다.
