# Antigravity IDE

Antigravity IDE는 에이전트 중심 시대에 누구나 개발할 수 있도록 진화시키는 에이전트 개발 환경입니다. 에이전트를 개발 워크스페이스에 직접 통합하여 에디터, 터미널 및 브라우저에서 자율적으로 작동하는 데 필요한 도구를 제공하며, 작업을 통한 검증과 상위 레벨의 사용자 소통 및 아티팩트를 강조합니다. 이 기능을 통해 에이전트는 더 복잡하고 종단 간(end-to-end) 소프트웨어 작업을 계획하고 실행할 수 있으며, 기능 빌드, UI 반복 및 버그 수정에서 연구 및 보고서 생성에 이르기까지 개발의 모든 측면을 향상시킵니다.

## 핵심 용어

- **에이전트 (Agent)**: Antigravity 내의 주요 AI 모달리티입니다. 사용자는 에디터 내에서 에이전트와 긴밀하게 협업할 수 있을 뿐만 아니라, 여러 코드베이스에 걸쳐 여러 에이전트를 작동시킬 수도 있습니다.
- **탭 (Tab)**: Antigravity 내의 또 다른 AI 모달리티로, 특히 에디터 영역의 텍스트 에디터 부분에 해당합니다. 탭은 더 강력한 자동 완성 도우미입니다.
- **아티팩트 (Artifacts)**: 에이전트가 작업을 완료하거나 인간 사용자에게 자신의 성과를 전달하기 위해 생성하는 모든 것을 아티팩트로 정의합니다. 여기에는 풍부한 마크다운 파일, diff 뷰, 아키텍처 다이어그램, 이미지, 브라우저 기록 등이 포함됩니다.
