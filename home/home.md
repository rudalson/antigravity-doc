# Google Antigravity에 오신 것을 환영합니다

## 작업 환경 선택하기

Google Antigravity는 특정 개발 워크플로우에 맞춘 다양한 제품 환경(Surface)을 제공합니다. 개발자님의 필요에 가장 잘 맞는 인터페이스를 선택하세요:

### Antigravity 2.0

에이전트를 위한 단독 데스크톱 커맨드 센터입니다. 프로젝트 내에서 에이전트를 시작하고, 여러 워크스페이스와 워크트리를 교차해 작업하며, 병렬 로컬 서브에이전트를 사용하여 복잡한 작업을 오케스트레이션합니다.

* **주요 기능**: 비동기 작업 관리, 예약된 작업(Cron 사이드카), 음성 전사.  
* **시작하기**: [시작 가이드](/docs/getting-started) 읽기

### Antigravity CLI

키보드 중심의 가벼운 터미널 사용자 인터페이스(TUI) 환경입니다. 데스크톱 앱과 동일한 핵심 에이전트 기능을 터미널 워크플로우에 직접 제공하여 빠른 상호작용 및 SSH 세션에 완벽하게 적합합니다.

* **주요 기능**: 고속 프롬프트 단축키, 사용자 정의 키 바인딩, 병렬 서브에이전트 관리.  
* **시작하기**: [CLI 빠른 개요](/docs/cli-overview) 살펴보기

### Antigravity SDK

에이전트 배포를 완벽하게 제어하고자 하는 연구자와 개발자를 위한 프로그래밍 방식의 Python 프레임워크입니다. Antigravity Harness를 기반으로 에이전트를 맞춤 빌드하고, 사용자 정의 도구를 등록하고, 수명 주기(lifecycle) 훅을 구현합니다.

* **주요 기능**: 선언적 보안 정책, 검사/결정/변환(inspect/decide/transform) 훅, 프로그래밍 방식의 서브에이전트 생성.  
* **시작하기**: [SDK 개요](/docs/sdk-overview) 검토하기

### Antigravity IDE 

모든 기능을 갖춘 AI 기반 개발자 환경입니다. 강력하고 긴밀하게 통합된 코딩 에이전트, 깊은 컨텍스트 인식, MCP 및 스킬과 같은 도구 등을 통해 일상적인 코딩을 표준화하세요.

* **시작하기**: [IDE 시작 가이드](/docs/ide-getting-started) 읽기

### 핵심 에이전트 기능

모든 Antigravity 환경은 Gemini 모델과 함께 공동 학습되고 고도로 최적화된 공유 에이전트 하네스(Harness) 상에서 작동합니다:

* **Gemini 3.5 Flash**: 최첨단(SOTA) 속도, 추론, 컨텍스트 창 용량으로 모든 로컬 에이전트에 강력한 성능을 제공합니다.  
* **[비동기 서브에이전트](/docs/subagents)**: 메인 에이전트가 흐름을 방해하지 않고 동시 백그라운드 작업을 병렬 서브에이전트에 위임할 수 있도록 지원합니다.  
* **[시각적 아티팩트](/docs/artifacts)**: 에이전트 출력 결과(계획, 코드 diff, 브라우저 녹화본)를 고해상도 시각적 보고서로 추적하고 확인하여, 진행 상황을 단계별로 파악할 수 있도록 돕습니다.  
* **[보안 고려 설계(Security by Design)](/docs/permissions)**: 안전한 기본값, 로컬 프록싱, 세분화된 도구 승인 단계를 통한 안전한 로컬 실행을 보장합니다.  
* **[Google 통합](/docs/build-with-google)**: Google 전반의 제품 팀과 협력하여 Google 플랫폼에서의 개발을 원활하게 만드는 엄선된 스킬 번들, MCP 서버 및 확장 프로그램을 제공합니다.  
  * **Android**: 에디터 확장 프로그램, CLI 통합 및 Android 개발자 스킬.  
  * **Firebase**: Firebase Firestore, Cloud Functions 등을 위한 엄선된 스킬.  
  * **Web**: 자율적인 브라우저 탐색을 위한 Chrome 및 웹 MCP 서버.  
  * **Science**: 과학 분야의 워크플로우를 가속화하기 위한 DeepMind의 전문 생물학 및 화학 스킬.  
  * **AGY SDK**: 사용자의 워크플로우에 맞춤화된 AI 에이전트를 빌드하기 위해 Antigravity SDK를 사용할 수 있도록 에이전트 역량을 최적화하는 스킬.
