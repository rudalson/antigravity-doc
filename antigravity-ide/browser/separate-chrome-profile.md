# 별도의 Chrome 프로필

일반 웹 브라우징으로부터 브라우저를 격리하기 위해, 에이전트는 [별도의 Chrome 프로필](https://support.google.com/chrome/answer/2364824)에서 작동합니다.

Chrome 프로필은 격리되어 있으므로 일반 브라우징 프로필의 쿠키나 로그인 정보는 공유되지 않습니다. 하지만 모든 로그인은 지속적으로 유지되므로 나중에 브라우저를 열 때마다 모든 계정이 그대로 유지됩니다.

이 프로필을 실행하는 동안 일반 Chrome이 열려 있었다면 별도의 독(dock) 아이콘으로 표시되며 별개의 애플리케이션으로 간주됩니다. 이전에 Chrome이 열려 있지 않았다면 이 애플리케이션은 기본 프로필과 동일하게 보입니다. 기본 프로필로 돌아가려면 애플리케이션을 종료하고 Chrome을 다시 실행해야 합니다.

브라우저 프로필이 생성될 위치를 변경하려면 브라우저 섹션에서 다음 설정을 수정할 수 있습니다.
