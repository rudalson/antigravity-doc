# 워크스루 (Walkthrough)

에이전트는 작업 구현을 완료한 후 워크스루 아티팩트를 생성합니다. 이 유형의 아티팩트에는 진행된 대화 중에 수행된 변경 사항에 대한 간략한 요약이 포함되어 사용자에게 일어난 일을 상기시켜 줍니다. 이는 에이전트가 모든 변경 사항을 처리한 후, 사용자가 작업 과정을 처음부터 끝까지 직접 따라가지 않았더라도 코드베이스의 현재 상태를 신속하게 파악할 수 있는 유용한 방법입니다.

![워크스루](assets/image/docs/artifacts/artifact-walkthrough.png)

브라우저 작업의 경우, 워크스루에 에이전트가 브라우저에서 구축하거나 생성한 내용의 스크린샷과 화면 녹화본이 포함되는 경우가 많습니다!

![이미지가 포함된 워크스루](assets/image/docs/artifacts/artifact-walkthrough-image.png)
