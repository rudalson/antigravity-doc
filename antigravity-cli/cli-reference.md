# CLI 참조

모든 TUI 슬래시 명령, 기본 키보드 단축키 및 JSON 구성 매개변수를 나열하는 요약 테이블을 탐색합니다.

## 핵심 슬래시 명령

프롬프트 상자 안에 `/`를 입력하여 자동 완성 명령 선택 메뉴를 엽니다.

| 명령 | 범주 | 별칭 | 실행 목적 |
| :-------------------- | :------------- | :------------------------- | :--------------------------------------------------------------------------------------- |
| **`/add-dir <path>`** | 유틸리티 | — | 활성 워크스페이스에 디렉터리 경로를 추가합니다. |
| **`/agents`** | 도구 및 작업 | — | 백그라운드 서브에이전트를 모니터링하기 위해 에이전트 관리자 패널을 엽니다. |
| **`/btw <query>`** | 유틸리티 | — | 주 대화를 중단하지 않고 백그라운드에서 사이드 질문을 합니다. |
| **`/clear`** | 유틸리티 | — | 터미널을 지우고 활성 대화 컨텍스트를 재설정합니다. |
| **`/config`** | 구성 | `/settings` | 대화형 설정 편집기 오버레이를 엽니다. |
| **`/diff`** | 유틸리티 | — | 수정된 모든 워크스페이스 파일의 통합 diff 표현을 보여줍니다. |
| **`/exit`** | 핵심 | — | TUI 세션을 종료하고 호스트 쉘을 복원합니다. |
| **`/fast`** | 구성 | — | 빠른 작업을 위해 빠른 모드(추론 계획 우회)를 활성화합니다. |
| **`/fork`** | 대화 | `/branch` | 현재 대화 스레드를 새로운 병렬 세션으로 복제합니다. |
| **`/hooks`** | 도구 및 작업 | — | 활성 실행 전(pre-flight)/포맷 후(post-format) 스크립트 훅을 탐색합니다. |
| **`/keybindings`** | 구성 | — | 대화형 키보드 단축키 편집기를 엽니다. |
| **`/logout`** | 계정 | — | 프로필 연결을 끊고 보안 키링에서 인증 토큰을 제거합니다. |
| **`/mcp`** | 도구 및 작업 | — | Model Context Protocol(MCP) 서버 관리자를 엽니다. |
| **`/model`** | 구성 | — | 선호하는 추론 모델을 선택합니다(세션 간에 유지됨). |
| **`/open <path>`** | 유틸리티 | — | 기본 시스템 편집기 내부에서 해당 경로를 강제로 엽니다. |
| **`/permissions`** | 구성 | — | 전역 권한 사전 설정(`request-review`, `always-proceed`, `strict`) 간에 전환합니다. |
| **`/planning`** | 구성 | — | 복잡한 엔지니어링 작업을 위해 다단계 계획 생성 모드를 활성화합니다. |
| **`/rename <name>`** | 대화 | — | 현재 세션 스레드의 이름을 변경합니다. |
| **`/resume`** | 대화 | `/switch`, `/conversation` | 이전 스레드를 선택하고 로드하기 위해 대화 선택기 오버레이를 엽니다. |
| **`/rewind`** | 대화 | `/undo` | 대화 기록을 이전 메시지로 되돌립니다. |
| **`/skills`** | 도구 및 작업 | — | 로드된 로컬 및 전역 에이전트 스킬(Agent Skills)을 탐색합니다. |
| **`/statusline`** | 구성 | — | 상태 표시줄 사용자 정의 오버레이를 엽니다. |
| **`/tasks`** | 도구 및 작업 | — | 백그라운드 쉘 실행 로그를 모니터링하기 위해 작업 관리자 패널을 엽니다. |
| **`/title [on/off]`** | 구성 | — | 터미널 창 제목 업데이트를 토글하거나 설정합니다. |
| **`/usage`** | 유틸리티 | — | 터미널 내에서 오프라인 개발자 도움말 설명서를 실행합니다. |

## 기본 단축키

전역, 프롬프트, 탐색 및 승인 작업을 매핑하는 키보드 단축키 명령입니다.

### 전역 제어

이 단축키는 현재 어떤 패널, 오버레이 또는 프롬프트에 포커스가 있는지에 관계없이 항상 활성화되어 있습니다.

| 키 | TUI 명령 | 작업 동작 |
| :----------- | :----------------- | :------------------------------------------------------------------------- |
| **`Esc`** | `cli.escape` | 활성 패널을 닫거나, 활성 스트림을 중단하거나, 빈 프롬프트를 지웁니다. |
| **`Ctrl+C`** | `cli.exit` | CLI 세션을 종료합니다(에이전트가 작업 중인 경우 확인 프롬프트가 표시됨). |
| **`Ctrl+L`** | `cli.clear_screen` | 시각적 터미널 버퍼를 새로 고치고 지웁니다. |

### 프롬프트 포커스 키

이 키들은 프롬프트 상자 안에 지침을 작성할 때 활성화됩니다.

| 키 | TUI 명령 | 작업 동작 |
| :------------------------------- | :------------------------- | :------------------------------------------------------------------------- |
| **`Enter`** | `prompt.submit` | 프롬프트 또는 활성 메뉴 선택 항목을 에이전트에게 제출합니다. |
| **`Shift+Enter`** / **`Ctrl+J`** | `prompt.newline` | 제출하지 않고 깨끗한 새 줄을 삽입합니다. |
| **`Ctrl+V`** | `prompt.paste` | 그래픽 미디어 파일 또는 클립보드 블록을 프롬프트에 붙여넣습니다. |
| **`Ctrl+O`** | `prompt.toggle_trajectory` | 세부 도구 추론 출력을 확장하거나 축소합니다. |
| **`Ctrl+R`** | `prompt.open_review` | 아티팩트 검토 패널을 엽니다. |
| **`Ctrl+G`** | `prompt.external_editor` | 프롬프트를 구성하기 위해 기본 `$EDITOR` 쉘을 실행합니다. |
| **`Alt+J`** | `prompt.teleport_agent` | 확인을 기다리는 다음 서브에이전트로 포커스를 즉시 전환합니다. |
| **`Ctrl+K`** | `prompt.fast_approve` | 상태 알림에 나열된 보류 중인 서브에이전트 작업을 즉시 승인합니다. |
| **`Ctrl+A`** | `prompt.cursor_start` | 프롬프트 삽입 커서를 줄의 시작 부분으로 이동합니다. |
| **`Ctrl+E`** | `prompt.cursor_end` | 프롬프트 삽입 커서를 줄의 끝 부분으로 이동합니다. |
| **`Ctrl+Z`** | `prompt.undo_text` | 마지막 편집을 되돌립니다. |
| **`Ctrl+Shift+Z`** | `prompt.redo_text` | 마지막으로 실행 취소된 텍스트 작업을 다시 실행합니다. |

### 탐색 및 스크롤

선택 패널, 메뉴 및 스크롤 가능한 텍스트 상자 내부에서 사용됩니다.

| 키 | TUI 명령 | 작업 동작 |
| :------------------------- | :------------------------------------- | :----------------------------------------------------------------- |
| **`↑`** / **`↓`** | `navigation.up` / `navigation.down` | 강조 표시된 선택 항목을 한 항목씩 위 또는 아래로 스크롤합니다. |
| **`PgUp`** / **`Shift+↑`** | `navigation.page_up` | 활성 텍스트 뷰포트를 한 페이지 블록씩 위로 스크롤합니다. |
| **`PgDn`** / **`Shift+↓`** | `navigation.page_down` | 활성 텍스트 뷰포트를 한 페이지 블록씩 아래로 스크롤합니다. |
| **`←`** / **`→`** | `navigation.left` / `navigation.right` | 다중 페이지 구조(예: 세션 선택기) 내에서 페이지를 전환합니다. |
| **`Tab`** | `navigation.tab` | 강조 표시된 슬래시 명령 자동 완성 옵션을 확인합니다. |

### 도구 승인

확인 프롬프트 중에 활성화됩니다.

| 키 | TUI 명령 | 작업 동작 |
| :----- | :------------ | :---------------------------------------------------------------------------------------- |
| **`y`** | `confirm.yes` | 제안된 도구, 명령 또는 활성 아티팩트를 승인합니다. |
| **`n`** | `confirm.no` | 제안된 도구, 명령 또는 활성 아티팩트를 거부합니다. |
| **`A`** | `—` | (검토 패널 내에서) 생성된 모든 아티팩트를 한 번에 승인합니다(내장 단축키). |

## 구성 키 (`settings.json`)

주요 설정 키 이름, 데이터 유형, 시스템 기본값 및 예상 매개변수입니다.

| 옵션 키 이름 | 값 유형 | 시스템 기본값 | 매개변수 특징 및 옵션 |
| :---------------------------- | :--------- | :------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **`colorScheme`** | string | `"terminal"` | 색상 테마: `"light"`, `"solarized light"`, `"colorblind-friendly light"`, `"dark"`, `"solarized dark"`, `"colorblind-friendly dark"`, `"tokyo night"` 또는 `"terminal"` (기본 쉘 색상 상속). |
| **`altScreenMode`** | string | `"default"` | 화면 버퍼 사용: `"default"` (기본 터미널 인라인), `"always"` (보조 화면 깜박임 없는 버퍼). |
| **`toolPermission`** | string | `"request-review"` | 전역 안전 사전 설정: `"request-review"` (쓰기/bash/web 도구에 대한 확인 메시지 표시), `"proceed-in-sandbox"` (샌드박스 내 자동 진행), `"always-proceed"` (확인 메시지 없음) 또는 `"strict"` (모든 비읽기 도구에 대해 확인 메시지 표시). |
| **`artifactReviewPolicy`** | string | `"asks-for-review"` | 코드 검토 정책: `"asks-for-review"` (코드를 작성하기 전에 항상 확인 메시지 표시), `"agent-decides"` (동적으로 확인 메시지 표시) 또는 `"always-proceed"` (확인 메시지 없음). |
| **`notifications`** | boolean | `false` | 작업 완료 시 시스템 데스크톱 및 터미널 벨 벨소리 알림을 생성합니다. |
| **`showTips`** | boolean | `true` | 생성 단계 중에 프롬프트 패널 위에 유용한 에이전트 팁을 표시합니다. |
| **`showFeedbackSurvey`** | boolean | `true` | 활성 작업 완료 시 주기적으로 품질 피드백 설문조사를 표시합니다. |
| **`editor`** | string | `"auto"` | 대상 텍스트 편집기 유틸리티: `"auto"` (시스템 `$EDITOR` 참조), `"vim"`, `"emacs"` 또는 사용자 정의 텍스트 레이블. |
| **`allowNonWorkspaceAccess`** | boolean | `false` | 에이전트의 파일 읽기 및 쓰기 도구가 인식된 Git/워크스페이스 루트 외부로 탐색할 수 있도록 허용합니다. |
| **`enableTerminalSandbox`** | boolean | `false` | 에이전트가 실행하는 모든 로컬 실행 명령을 OS 격리 링으로 제한합니다. |
| **`enableTelemetry`** | boolean | `true` | 도구 신뢰성을 개선하기 위해 메트릭 수집 및 충돌 로그 스트리밍을 허용합니다. |
| **`verbosity`** | string | `"high"` | 시각적 세부 정보 수준: `"high"` (전체 추론 과정 및 도구 출력 렌더링) 또는 `"low"` (최소한의 시각적 진행률 표시기만 표시). |
| **`runningLightSpeed`** | string | `"medium"` | 진행 상황을 보여주는 러닝 라이트 애니메이션 속도: `"fast"`, `"medium"`, `"slow"` 또는 `"off"`. |
