# 스크린샷 (Screenshots)

브라우저 서브 에이전트는 페이지 상태에 대한 사용자의 검토가 필요할 때 열려 있는 페이지나 페이지의 요소의 스크린샷을 찍을 수 있습니다. 이는 에이전트에게 제공되는 도구로 구동되며, 사용자가 직접 에이전트에게 페이지 스크린샷을 찍도록 요청할 수도 있습니다.

![브라우저 스크린샷 캡처 도구](assets/image/docs/artifacts/browser-screenshot-capture.png)

모든 스크린샷은 이미지 아티팩트로 저장되며, 에이전트에게 피드백을 주기 위해 댓글을 작성할 수 있습니다.

![브라우저 스크린샷 아티팩트](assets/image/docs/artifacts/browser-screenshot-artifact.png)
