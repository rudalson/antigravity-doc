# 변경 사항 검토

에이전트가 대화 내에서 코드를 작성하기 시작하면 에이전트 패널의 [하단 툴바](docs/agent-side-panel)에 `Review Changes` 섹션이 표시됩니다. 이를 클릭하면 에디터 내에 창이 열려 대화 중에 사용자와 에이전트가 수행한 모든 변경 사항을 스크롤하여 확인할 수 있습니다.

![에디터 변경 사항 검토](assets/image/docs/editor/review_changes_editor.png)

아티팩트와 마찬가지로 파일 diff 중 하나에 의견을 작성하여 에이전트와 대화할 수 있습니다.
